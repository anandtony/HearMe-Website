HearMe

Assistive communication web application

HearMe is a simple web-based system designed to support deaf and hearing-impaired users. The application helps users communicate using speech-to-text and hand gesture detection through a web browser.

System Architecture

The frontend of HearMe is built using HTML, CSS, and JavaScript. Real-time processing is performed in the browser, while a Node.js backend is used to store and manage data.
